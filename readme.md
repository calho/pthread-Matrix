to make:
$ make

to run matrixgen
$ ./matrixgen

to run matrixgen with parameters
generate two 100 by 100 matrices with elements between -10 and 10, print the result and store it in "data_input"
$ ./matrixgen -s 100 -b 10 -p

to run main
$ ./main [# of threads]

to run serialtester
$ ./serialtester

to run check
$ ./check.sh
